Project on DataSport
Applied Data Analysis

Abstract

This project aimed to focus on Lausanne Marathon and more precisely on the analysis of the different runs (10km, semi-marathon and marathon). To do so, we manipulated available data - from 1999 to 2016 - and used different information stored about the runners and provided by DataSport.

First part: analysis of Lausanne Marathon of 2016

In the present part, we studied performance of runners according to different parameters like gender or age. We also analyzed the behavior of runners in different contexts (speed following the type of running, speed of male and female runners, etc.) and checked if there are some statistical differences or some patterns.

Second part: analysis over the years (1999-2016)

This part of the study focused on all runnings of Lausanne Marathon, between 1999 and 2016. Among the analysis we planned to do, we included a presentation of the evolution of the performance following the years. We also verified if being in group ensure better score/time and we checked, for identified groups, if we can identify an improvement of performance over the years. We also conducted basic analysis as in the first part.

Third part: analysis of some runners

In the third and last part, we selected sufficient number of runners to conduct statistical analysis, like the evolution of individual performance (type of chosen running, average speed, etc.).

Data description

This project uses data from the www.datasport.com website, which contains sport performance of athletes over time.

The website contains some data of races from the entire Switzerland. Moreover the website contains data throughout the years. This was useful for our study of the evolution of sport in our society and especially running, which has grown exponentially the last decade.

How to use visualisations

At some points in the website, we made some dynamic charts that you can manipulate to see what you want. See below some useful tips:

  1. Select one value in every field regarding what you want to see, and wait few seconds that the graph loads
  2. These two options allow to display a comparison of all lines or just information about the point when you make an hover with the mouse
  3. Make an auto scale or reset it to the default one
  4. Zoom, move, zoom in, zoom out in the graph with your mouse
  5. Select which values to display by clicking on it to enable/disable
  6. Interact with the graph to get more information
